# Synthetic Brand Reality Prototype

Prototype app for luxury marketing and security teams monitoring unauthorized AI-generated brand content.

## Run Locally

```bash
npm start
```

Open `http://127.0.0.1:5173`.

The prototype has no package dependencies. It uses static HTML, CSS, JavaScript, and synthetic fixture data.

## Included Workflows

- Command center with risk metrics and priority incidents.
- Live scan feed with platform and severity filtering.
- Archive fingerprint view with verified synthetic assets and explainable model signals.
- Case review board with owner and status controls.
- Takedown draft workflow with evidence references.
- Executive brief with current trust-risk summary.

All names, brands, social handles, URLs, incidents, media previews, and model outputs are fictional.
